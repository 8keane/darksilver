Welcome to the DarkSilver project

This project aims at creating a web application for Research Department, JKUAT for easier management of manuscripts published in the Journal of Agriculture, Science and Technology. The web technologies used for the project are: *HTML5 *CSS *Javascript *JSP *Servlets *MySQL

To make your contribution, simply fork the project, clone it, implement a new feature, then push it to your github account and make a pull request.
